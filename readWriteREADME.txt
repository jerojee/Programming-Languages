The reader writer program source code is contained in the "ReaderWriter.java" file.

To compile the java program on empress, enter "javac" followed by the filename and it's .java extension.
The command to compile this program is as follows:
   javac ReaderWriter.java

After compiling the program, to run it, enter "java" followed by the class name in the source file.
The command to run it after compiling is as follows:
   java ReaderWriter
 
The program will then display multiple threads, reader threads, and writing threads, as well as their status.
